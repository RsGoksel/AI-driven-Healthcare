# Abdomen Multi-Disease Detection

Computer-vision project for detecting multiple acute abdominal conditions from CT imaging, developed for **TEKNOFEST 2022 — Healthcare AI Competition**, category *"Computer Vision-Based Disease Detection for the Abdomen Region"* (Bilgisayarlı Görüyle Abdomen (Karın) Bölgesi İçin Hastalık Tespiti Kategorisi).

## Competition context

Participants were given contrast and non-contrast abdominal CT series from emergency-department admissions presenting with abdominal pain, and asked to detect which of **six clinical conditions** was present:

1. Acute appendicitis
2. Acute cholecystitis
3. Acute pancreatitis
4. Kidney / ureteral stone
5. Acute diverticulitis
6. Aortic aneurysm / rupture / dissection

Source: [TEKNOFEST 2022 Sağlıkta Yapay Zeka Yarışması — Abdomen Kategorisi](https://semruk.tech/teknofest-2022-saglikta-yapay-zeka-yarismasi-bilgisayarli-goruyle-abdomen-karin-bolgesi-icin-hastalik-tespiti-kategorisi/), [TEKNOFEST — Sağlıkta Yapay Zeka Yarışması](https://teknofest.org/tr/yarismalar/saglikta-yapay-zeka-yarismasi/).
